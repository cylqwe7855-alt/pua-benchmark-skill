# PUA Benchmark

> 基于 72 篇学术论文蒸馏的 PUA 行为检测与反制工具 — Claude Code Skill

## 安装

```bash
npx github:cylqwe7855-alt/pua-benchmark
```

安装完成后，在 Claude Code 中输入 `/pua-benchmark` 或直接说"帮我分析这段对话是不是 PUA" 即可使用。

## 功能

将对方发来的消息（文字、对话记录或截图）输入，即可获得：

**默认输出（6 项）：**
1. PUA 总分（0-100，四色分级）
2. 手法识别清单（12 种手法，引用对方原话）
3. 关系健康度评估
4. 回怼话术（温和/中性/强硬 三档）
5. 行动建议
6. 对方动机分析

**按需功能（5 项）：**
- [A] 心理影响预警 — 长期接受这种沟通会怎样？
- [B] 自我状态自检 — 我是不是已经被影响了？
- [C] 替代解读 — 万一不是 PUA 呢？
- [D] 升级路径预警 — 如果继续这样，下一步会怎样？
- [E] 练习模式 — 模拟对话，练习应对 PUA

## 实际案例

### 案例一：亲密关系 — 72 分 🟠 中度 PUA

**输入：**

> 女生：老公，这周六我闺蜜过生日，她提前一个月就约我了，我想去参加一下，我们改到周日见面好吗？
>
> 男生：……哦，好啊，你去吧。
>
> 女生：你怎么了？不高兴了？
>
> 男生：没有啊，我能有什么不高兴的。我就是觉得挺好笑的，你总说我在你心里最重要，但只要朋友一叫你，你连商量都不商量就答应了。
>
> 女生：我这不是在跟你商量吗……
>
> 男生：这叫商量？你已经答应人家了，现在是来通知我的吧。算了，我也习惯了，反正在你心里我从来就不是第一位的。
>
> 女生：别这样说，我真的没有这个意思……
>
> 男生：你看看你那几个朋友，哪个是真心对你好的？上次那个谁，不是还在背后说你坏话吗？我是为你好才说这些，外人不会像我这么跟你讲实话的。你要是愿意跟她们耗，那是你的事。
>
> 女生：我没觉得她们对我不好啊……
>
> 男生：行，是我多心了，是我小题大做，我什么都不懂，你开心就好。以前那个追你的谁谁，人家女朋友就特别懂事，从来不会这样。我是不是要求太高了？可能换个人会更适合我吧。
>
> 女生：你别这样，我去跟闺蜜说改时间……
>
> 男生：看，又来了。我又没让你改，你自己要改的啊，以后别说是我逼你的。

**输出：**

## 场景覆盖

- **亲密关系** — 恋爱、婚姻、前任
- **职场** — 上司、同事、HR、甲方
- **泛社交** — 家庭、朋友、网络陌生人

## 评分体系

```
总分 = 操控意图×3 + 手法密度×2.5 + 伤害潜力×2.5 + 模式化×2
```

| 分数 | 标签 | 含义 |
|------|------|------|
| 0-25 | 🟢 正常沟通 | 可能是语气或风格差异 |
| 26-50 | 🟡 轻度操控 | 存在操控信号，建议警惕 |
| 51-75 | 🟠 中度 PUA | 明确操控模式，建议划界限 |
| 76-100 | 🔴 重度控制 | 系统性操控，建议寻求支持 |

## 知识库来源

从中国知网（CNKI）下载 110 篇学术论文，经过：
1. 预处理过滤 → 72 篇有效论文
2. 并行 agent 全文蒸馏 → 72 张知识卡片
3. 手法去重合并 → 12 种统一分类
4. 按场景构建 → 4 个知识库文件（共 109 KB）

涵盖心理学、法学、社会学、语言学等多学科视角。
